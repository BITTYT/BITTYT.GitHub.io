  • dfs 搜索过程图论基础：图分为两种有向图和无向图无向图中有几条边连接该节点，该节点就有几度。有向图中，每个节点有出度和入度”出度：从该节点出发的边的个数。入度：指向该节点边的个数。“。无向图中，任何两个节点都是可以到达的，我们称之为连通图，反之称为非连通图。。有向图中，任何两个节点是可以相互到达的，我们称之为 强连通图。(任何两个节点是可以相互到达)。无向图中的极大连通子图称之为该图的一个连通分量。称为极大是因为如果此时加入任何一个不在图的点集中的点都会导致它不再连通,并不是因为他是最大的连通图，必须是极大联通子图才能是连通分量。一般使用邻接表、邻接矩阵 或者用类来表示表示图。dfs搜索原理：沿着一条路径去搜索，遇到搜索过的路径结束。代码：。void dfs(参数){。处理节点。dfs(图，选择的节点)；//递归。回溯。撤销处理结果。} （与回溯法相似） void dfs(参数) { if (终止条件) { 存放结果; return; } for (选择：本节点所连接的其他节点) { 处理节点; dfs(图，选择的节点); // 递归 回溯，撤销处理结果 } }
